1. Вношу изменения в файл на гитхабе

2. Проверяю идентичность
>git status

3. Просматриваю содержимое файла на локальном репозитории
 >cat first_task.txt

4. Загружаю изменения с удаленного
 >git fetch

5. Смотрю, на какое количество коммитов отстает локальный репозиторий
 >git status

6. Обновляю локальный репозиторий
 >git pull

7. Проверяю идентичность
 >git status
