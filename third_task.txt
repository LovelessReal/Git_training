1.Команда для просмотра наименования всех веток 

> git branch (локальные ветки)
> git branch -r (удаленные ветки)
> git branch -a (все ветки: локальные + удаленные)

2. Переключение на другую ветку (например, ветку test)
> git checkout test

3. Чтобы узнать название главной ветки
> git branch -r
или
> git remote show origin
Ветка, которая помечена как HEAD branch —  и есть главная
