1. Создаю репозиторий на гитхаб

2. Клонирую репозиторий на свой пк  
 >git clone <link> 

3. Проверяю идентичность локального и удаленного репозиториев
 >git status

4. Создаю файл first_task.txt 

5. >git add first_task.txt

6. Создаю коммит
>git commit -m "test commit 1"

7. Отправляю коммит на удаленный репозиторий
 >git push
